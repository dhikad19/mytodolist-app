## ToDo List App Project

This app good for organize, schedule, plan and analyze your projects easily with simple interface.

## This App Using

1. HTML
2. CSS
3. Python
4. Django

## Run the Program

Type in Terminal
```sh
   python manage.py runserver
   ```
